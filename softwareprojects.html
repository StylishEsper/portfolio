NFTY

Probably the most fun creation: an NFT generator named NFTY (pronounced Nifty). I developed this in Unity. Why Unity? Two reasons: I'm comfortable with the software, and I thought it would be awesome to be able to use Unity's built-in features on the NFTs! It currently only works in the Unity Editor.

I can create 2D and 3D NFTs in a variety of file formats, including video (which will also record sound). Mass-generating randomized NFTs with automatically generated .JSON files is possible as well! The example NFT looks like a pixel artist drew it, but it's just mostly 3D objects on a canvas, and the canvas it exists on is pixelated through code. The pixel dimensions can be changed easily.

C# / UNITY
